# ML Final Exam - Diabetes Prediction System

## Overview
This project implements a complete Machine Learning pipeline for predicting diabetes using the Pima Indians Diabetes Dataset.

## Tasks Completed
1. **Data Loading** - Loaded dataset with 768 samples and 9 features
2. **Data Preprocessing** - 5 distinct preprocessing steps (handling zeros, outlier detection, feature engineering)
3. **Pipeline Creation** - sklearn Pipeline with StandardScaler and RandomForestClassifier
4. **Model Selection** - Random Forest Classifier (justified)
5. **Model Training** - Trained on 80% training data
6. **Cross-Validation** - 5-fold CV with mean/std scores
7. **Hyperparameter Tuning** - GridSearchCV optimization
8. **Best Model Selection** - Selected and saved best model
9. **Model Evaluation** - Accuracy, Precision, Recall, F1, ROC-AUC
10. **Gradio Interface** - User-friendly web interface
11. **Hugging Face Deployment** - Deployed to HF Spaces

## Files
- `ML_Final_Exam.ipynb` - Main notebook with all 11 tasks
- `app.py` - Gradio app for deployment
- `requirements.txt` - Python dependencies
- `model.pkl` - Trained model (generated by notebook)

## Usage

### Run Notebook
```bash
conda activate ai-ml
jupyter notebook ML_Final_Exam.ipynb
```

### Run Gradio Locally
```bash
conda activate web-dev
python app.py
```
